# gx
感想
流星划过夜空的那一闪耀是陨石的期待，蝴蝶破茧而出的那一起舞是毛毛虫的期待，雄鹰遨翔天际的那一矫健是雏鹰的期待……无论生命处何境地，我们心中总有一种期待。
　　来到北财已经一年多了，刚入学时的学习过程仿佛还是昨天，回头看看，看看刚入学时自己的样子，和现在有着多大的差别？
　　刚来到实训时，自己不免有些骄傲自满，总感觉不会差到哪去。后来开始做项目，面对不断深入的难度，渐渐磨灭了我的骄傲劲儿，我讨厌被别人落下，便开始暗中努力。
　　努力终究是有效果的，现在也没了刚开始骄傲，我明白自己不可能一辈子都那样，必须要很快速的融入新的环境，学会自学，现在的生活使我感到了愉快，教学方式等等，让我不禁全身心的投入到现在的学习任务中。原来也不爱承担事务，而现在不同了，做自己力所能及的事情，积极的配合老师。说实话，以前我的学习习惯不好，很容易就放弃，为了以后，便下定决心，把自己得改一改。当然也要注意形象，培养好自己的自学能力，下定好的决心不能轻易放弃，，说句实话，跟以前有很大的差别，但我会努力。
不是所有的努力都会有结果，但只求无愧于心。有很多事真得就那么不容易，但不管多难都很值得去做，至少曾经尝试过，付出过，也不会觉得有遗憾，付出和回报总是成正比的。一个人总都得向往着些什么。
